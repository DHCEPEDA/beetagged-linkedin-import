# BeeTagged - Google Play Store Upload Guide

## App Information
- **App Name**: BeeTagged
- **Package Name**: com.beetagged.app
- **Version**: 1.0.0
- **Category**: Productivity

## Description for Play Store
BeeTagged transforms contact management through intelligent social media integration. Connect Facebook and LinkedIn to automatically enhance contacts with smart tags based on work, education, location, and interests.

**Key Features:**
- Smart contact enhancement from social profiles
- Facebook friends integration
- LinkedIn connections import
- Advanced tag-based search and filtering
- Automatic categorization by workplace, education, location
- Offline access to enhanced data

## Required Permissions
- `READ_CONTACTS` - Access phone contacts for enhancement
- `WRITE_CONTACTS` - Update contacts with social data
- `INTERNET` - Connect to social media APIs
- `ACCESS_NETWORK_STATE` - Check connectivity

## Upload Steps

### 1. Google Play Console Setup
1. Go to [Google Play Console](https://play.google.com/console)
2. Create developer account ($25 one-time fee)
3. Complete account verification

### 2. Create App
1. Click "Create app"
2. App name: **BeeTagged**
3. Default language: **English (United States)**
4. App or game: **App**
5. Free or paid: **Free**

### 3. Upload for Internal Testing
1. Navigate to **Testing â†’ Internal testing**
2. Click "Create new release"
3. Upload the Android app bundle (AAB file)
4. Add release notes: "Initial BeeTagged release with contact enhancement"
5. Save and publish to internal testing

### 4. Configure App Details
- **Short description**: Smart contact management with social media integration
- **Full description**: Use the description above
- **App category**: Productivity
- **Content rating**: Everyone
- **Target age group**: Adults

### 5. Add Yourself as Tester
1. In Internal testing, go to **Testers** tab
2. Add your email address
3. Copy the internal testing link
4. Open link on Android device
5. Install directly from Play Store

## App Bundle Status
The Android app bundle is being generated. Once ready, you'll have:
- `beetagged-release.aab` (App bundle for Play Store)
- `beetagged-release.apk` (APK for direct installation)

## Contact Integration Features
Once installed, BeeTagged will:
- Access your phone's contact list
- Connect to Facebook and LinkedIn with your permission
- Enhance contacts with profile data
- Create searchable tags automatically
- Work offline after initial sync

## Privacy & Data Handling
- All social media connections require explicit user consent
- Contact data enhanced locally on device
- No contact information shared without permission
- Users can delete enhanced data anytime
